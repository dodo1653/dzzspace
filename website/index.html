Features

Built for the terminal

Everything you need for a premium multi-terminal workflow.

Grid-Based
Terminals

CSS grid layouts with presets from 1 to 9 panes. Split, resize, and rearrange terminals in a native-feeling grid that adapts to your workflow.

Workspace
Manager

Group terminals by project. Switch between workspaces with one click — each keeps its own grid layout, panes, and running terminal sessions.

Premium Terminal
Rendering

Glyph-aware canvas metrics, adaptive typography with JetBrains Mono, and xterm.js with canvas renderer. No DOM renderer — native terminal feel.

Keyboard
First

Ctrl+Shift+N for new terminal, Alt+1-9 to switch panes, Ctrl+B toggle sidebar, Ctrl+D close pane — everything at your fingertips.

Per-Pane
Profiles

CLI-aware profile detection. Each pane gets a unique nickname and color. Swap layouts, rename panes, and organize your terminal grid.

Layout
Presets

1, 2, 4, 6, or 9 pane layouts with a single click. The grid resizes instantly — no animation jank, no canvas corruption.
